# calculator
A simple calculator made using HTML, CSS and JavaScript

## Features
- Basic calculator operations:
    - Addition
    - Subtraction
    - Multiplication
    - Division
- Can only do operations on two operands at a time.
- AC - All Clear button to clear the screen.
- DEL - Acts as a 'backspace' button.
